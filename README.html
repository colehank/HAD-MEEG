This repository provides the codebase for preprocessing, validation, and visualization associated with the A large-scale MEG and EEG dataset for human action recognition.

Installation

The code is tested with Python 3.11+.

Option 1: Using uv (recommended)

This project uses uv for Python dependency management.
To install uv, see the official docs: Installation.

git clone https://github.com/colehank/HAD-MEEG.git
cd HAD-MEEG
uv sync

Option 2: Using pip (without uv)

You can still install the dependencies with pip. The code is tested with Python 3.11+.

git clone https://github.com/colehank/HAD-MEEG.git
cd HAD-MEEG

# using venv
python -m venv .venv
source .venv/bin/activate # On Windows: .venv\Scripts\activate

# or using conda
# conda create -n had-meeg python=3.11
# conda activate had-meeg

pip install -r requirements.txt

Configuration

  1. copy .env_example to .env:
cp .env_example .env
  1. Edit .env and set MEEG_BIDS_ROOT to the path of the M/EEG BIDS root directory on your machine. The .env file is loaded at runtime to locate the dataset.
